# Potion Game

## The Big Idea or Goal of the Project
Potion Game is a two-player strategy game inspired by Russian Roulette. Players take turns choosing whether to drink or pass a mysterious potion. Some potions are poisonous. The goal is to survive longer than your opponent by making smart and strategic decisions.

## Unique Features of the Project
- Players must balance risk and reward in every turn  
- Drinking a harmless potion allows the player to continue their turn  
- Each player has limited lives, so decisions matter  
- The game includes psychological elements such as bluffing and pressure  
- Simple rules with deeper strategy over time

cd softdes-finalproject-teamSD

---

### Step 4: Run the game
python main.py

If you have issues running the game, make sure:
- You are using a supported Python version (3.10–3.12)
- pygame is installed correctly
- Your virtual environment (if used) is activated
